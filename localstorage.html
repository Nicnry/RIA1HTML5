Dans cet exercice vous apprendrez à utiliser le localstorage présent sur tous les navigateurs actuels.
Le storage contient un array d'objets représentant des utilisateurs sous cette forme : { name: [username] }. Cet array est stocké sous la clé "users"
Le but de cet exercice est de récupérer ces utilisateurs, d'y ajouter Marcel et Jean-Jacques et de remettre le nouvel array dans le storage
L'ordre des objets dans l'array n'a pas d'importance
